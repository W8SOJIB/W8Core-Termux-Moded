# Banner Configuration for Termux

Customize Termux with W8Core-Termux-Moded Banner

**Package:** core-termux (banner config)  
**Author:** DevCoreX  
**Repository:** https://github.com/W8SOJIB/W8Core-Termux-Moded  
**Type:** Termux UI customization  
**License:** MIT

## Description

Configure a custom banner for Termux using W8Core-Termux-Moded. This package allows you to set a personalized welcome message that appears when you start Termux, enhancing your terminal experience.

## Dependencies

- Termux (base installation)

## Install

```bash
core install ui --banner
```

## Uninstall

```bash
core uninstall ui --banner
```

## Update

```bash
core update ui --banner
```

## Notes

- Source file: `~/.local/share/core-termux/assets/banner/devcorex.txt`
- removes Termux welcome message: motd
- Restart Termux to apply changes
